# הדר | Sashiko & Fine Line Studio
A personal portfolio website built with HTML, CSS, and JavaScript.

## About
This is the personal website of Hadar — a sashiko embroidery artist and fine line tattoo artist.  
The site presents two distinct creative worlds under one roof, with an animated transition between them.  
Built from scratch with a minimal, craft-inspired aesthetic featuring live-drawn embroidery patterns, botanical ink illustrations, and a custom cursor.

## Sections
- **Sashiko World** — Courses, workshops, and embroidery kits
- **Tattoo World** — Fine line portfolio and available flash designs
- **Shop** — Handmade products, prints, and accessories
- **About** — Background and contact

## How to Run
No installation needed. Just open `index.html` in any browser.

Or clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO
open index.html
```

> For best results, use a local server (e.g. VS Code Live Server) to avoid browser security restrictions on local files.

## Tech Stack
- HTML5 + Canvas API
- CSS3 (custom properties, animations, grid, flexbox)
- Vanilla JavaScript
- Google Fonts — Cormorant Garamond + Noto Serif Hebrew
